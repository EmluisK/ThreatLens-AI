# User Stories - SOC Analyst

---

**US-AN01 - Log In**
As a SOC Analyst, I want to log in with my email and password so that I can access my dashboard and start working on alerts.

---

**US-AN02 - View All Alerts**
As a SOC Analyst, I want to see a list of all alerts ordered by time so that I can quickly identify what needs attention.

---

**US-AN03 - View Alert Details**
As a SOC Analyst, I want to see the severity, status, message, and source of an alert so that I have enough context to decide how to handle it.

---

**US-AN04 - View the Original Log**
As a SOC Analyst, I want to open the log file that produced an alert so that I can read the surrounding lines and understand what happened.

---

**US-AN05 - Run AI Triage on an Alert**
As a SOC Analyst, I want to trigger an AI analysis on an alert so that I get a plain-language summary of what the alert means and concrete steps to address it.

---

**US-AN06 - View Triage History**
As a SOC Analyst, I want to see a record of all previously triaged alerts including who triaged them and what the AI returned so that I can refer back to past decisions.

---

**US-AN07 - View IoT Alert Details**
As a SOC Analyst, I want to see the malware family label on alerts that came from IoT submissions so that I can distinguish them from alerts generated by log file analysis.
