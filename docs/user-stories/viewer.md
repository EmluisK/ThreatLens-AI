# User Stories - Viewer

---

**US-V01 - Log In**
As a Viewer, I want to log in with my email and password so that I can access the read-only dashboard.

---

**US-V02 - View Platform Statistics**
As a Viewer, I want to see a summary of total alerts broken down by status (open, triaged, resolved) so that I have a general sense of the platform's current state.

---

**US-V03 - View Processed Alerts**
As a Viewer, I want to see a list of alerts that have been triaged or resolved so that I can monitor what threats have been identified and acted on.

---

**US-V04 - View Alert Severity and Status**
As a Viewer, I want to see the severity level and current status of each processed alert so that I can understand how serious the flagged issues are.
